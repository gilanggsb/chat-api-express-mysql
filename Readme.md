# Chat API with Express Mysql
    create chat api with express mysql
# Project Setup
1. Create a `.env` file in the root of your project and insert
your key/value pairs in the following format of `KEY=VALUE`:

    ```sh
    HOST=''
    USER_DB=''
    PASSWORD_DB=''
    DATABASE=''
    JWT_SECRET = ''
    SALT_KEY = ""
    ```
2. npm install

# How to use
```bash
 npm start
```
